Download and Install

All files associated with the current release are here
For Automatic Updates:
  1. Download the AppImage for this release - TabloTools-{{page.version}}.AppImage
  2. Make TabloTools-{{page.version}}.AppImage executable and run it.
    Follow these quick instructions if you're unsure how to do that.
Updates will happpen the next time the app is opened after a new release is made.

Standard packages:
  • tablo-tools_{{page.version}}_amd64.deb (ubuntu, debian, etc).
  • tablo-tools-{{page.version}}.x86_64.rpm (centos, redhat, etc).
You will have to check back for new versions on your own.
For Automatic Updates:
  1. Download the installer for this release - TabloTools-Setup-{{page.version}}.exe
  2. Install TabloTools-Setup-{{page.version}}.exe
Updates will happpen the next time the app is opened after a new release is made.

Other packages:
  • TabloTools-{{page.version}}.msi
You will have to check back for new versions on your own.
For Automatic Updates: x86 Packages:
  • Classic (x86) .dmg - TabloTools-{{page.version}}.dmg
  • M1/M2 (arm64) .dmg - TabloTools-{{page.version}}-arm64-mac.dmg
Updates will happpen the next time the app is opened after a new release is made.
note: As of v0.3.0, the Mac packages above are signed. Please let us know if you run into issues.

Other packages:
